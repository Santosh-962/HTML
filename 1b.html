Working of Hyperlinks in HTML

The <a> tag is used to create hyperlinks. The href attribute specifies the destination URL, and the target attribute defines where to open the link.

Examples:

  • Open Wikipedia in the same tab (target="_self")
  • Open Google in a new tab (target="_blank")
  • Open Microsoft in parent frame (target="_parent")
  • Open GitHub in full window (target="_top")
